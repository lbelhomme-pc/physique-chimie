# ETAT_AVANCEMENT_SITE_REVISION_PC

Date : Mars 2026

Projet : Plateforme de révision interactive de physique-chimie (collège → terminale)

Auteur : Ludovic Belhomme

---

# 1. Vision du projet

Créer une plateforme éducative permettant aux élèves de :

* réviser les cours
* s'entraîner avec des exercices
* tester leurs connaissances avec des quiz
* mémoriser avec des flashcards
* suivre leur progression

Niveaux concernés :

* Collège : 6e / 5e / 4e / 3e
* Lycée : Seconde / Première / Terminale

  * enseignement scientifique
  * spécialité physique-chimie
