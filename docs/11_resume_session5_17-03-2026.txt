4. ESTIMATION — SESSIONS RESTANTES AVANT LANCEMENT
------------------------------------------------------------

Pour un site à 80%+ prêt au lancement :

SESSION 6 — Contenu 3ème (la plus grosse)          ~3-4h
  • Convertir tous les HTML de 3ème chimie en MDX+JSON
  • Convertir tous les HTML de 3ème physique en MDX+JSON
  • Vérifier le rendu KaTeX, les exercices, les quiz
  • Au moins 8-10 chapitres complets

SESSION 7 — SEO + Déploiement                      ~2h
  • Meta tags, sitemap.xml, robots.txt
  • Schema.org (EducationalContent)
  • Open Graph (partage réseaux sociaux)
  • Déployer sur Vercel ou Netlify
  • Nom de domaine
  • Le site est EN LIGNE

SESSION 8 — Outils & Méthodes + polish             ~2-3h
  • Page Outils (formulaires, convertisseur, calculatrice)
  • Page Mémorisation (Mega Flash + Mega Quiz fonctionnels)
  • Barre de recherche fonctionnelle
  • Derniers ajustements design
  • Tests mobiles

SESSION 9 — Auth + Paiement (optionnel au lancement) ~3h
  • Supabase : inscription, connexion, profil
  • Migration localStorage → base de données
  • Stripe : abonnement premium
  • Verrouillage contenu payant

SESSIONS 10+ — Post-lancement
  • Contenu autres niveaux (6e, 5e, 4e, lycée)
  • Laboratoire de simulations
  • Escape game
  • PWA / app mobile
  • Améliorations continues
