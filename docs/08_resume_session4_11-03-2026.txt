### B) Corrections anti-triche + input obligatoire

  QuizPlayer v3 → v4 :
  - Questions ET réponses mélangées à chaque tentative
  - Anti-triche : XP donnés 1 seule fois par jour par quiz
  - Bandeau informatif si déjà fait aujourd'hui
  - Bouton "Recommencer" = re-shuffle complet

  FlashcardsPlayer v2 → v3 → v4 :
  - Input texte OBLIGATOIRE (l'élève doit écrire avant de voir)
  - Comparaison côte à côte (sa réponse vs bonne réponse)
  - Auto-évaluation (2 boutons : Incorrect/Partiel ou Correct)
  - Mélange automatique au recommencement

  ExercicesPlayer v3 → v4 :
  - Input texte OBLIGATOIRE
  - Auto-évaluation 3 niveaux avec XP proportionnels :
    😰 Incorrect (+1 XP) / 😕 Partiel (+2 XP) / 😎 Correct (+3 XP)
  - Anti-triche persistant (localStorage, 1 seule récompense par exo)
  - Indicateur ✅ sur les exercices déjà consultés


### C) Étape 4 — Répétition espacée (SRS / Anki)

  Moteur SRS créé (srs.ts) :
  - Algorithme SM-2 simplifié
  - 4 boutons : 😰 Oublié / 😕 Difficile / 🙂 Bien / 😎 Facile
  - Intervalles progressifs : 1j → 3j → 7j → 18j → 45j → ... → 365j max
  - Stockage par carte dans localStorage
  - Méthodes : review(), getDueCards(), getNewCards(), getChapterStats()
  - Aperçu des intervalles sur chaque bouton

  FlashcardsPlayer v4 (refonte complète) :
  - Écran de choix : "Révision du jour" / "Nouvelles cartes" / "Toutes"
  - Stats SRS affichées (à revoir / nouvelles / maîtrisées)
  - 4 boutons Anki avec intervalle affiché sous chaque bouton
  - Option "Revoir les cartes oubliées" en fin de session
  - Barre de progression mémorisation


### D) Branchement dailyLogin + tracking cours lu

  DailyLoginTracker.tsx :
  - Composant invisible dans le BaseLayout
  - Déclenche engine.dailyLogin() 1 fois par jour
  - Gère le streak automatiquement
  - +10 XP connexion + bonus streak (7j=+50, 30j=+200, etc.)

  CoursTracker.tsx :
  - Wrapper autour du contenu MDX
  - Détecte scroll ≥ 90% → cours marqué comme lu
  - +10 XP (1 seule fois par chapitre)

  Pages chapitre v7 : intègrent CoursTracker autour du cours
