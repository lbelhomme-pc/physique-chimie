PRINCIPE
────────

La page d'accueil est le COCKPIT de l'élève.
En un coup d'œil il voit :
  - Où il en est (XP, rang, streak)
  - Ce qu'il doit faire aujourd'hui (révision du jour)
  - Ses objectifs de la semaine
  - Sa progression par niveau/matière
  - Ses stats récentes
  - Ses dernières récompenses
  - Un accès rapide à tout le contenu

L'accueil est DIFFÉRENT selon que l'élève est :
  - Nouveau (pas de données) → version "découverte"
  - Habitué (données existantes) → version "tableau de bord"

============================================================
  PAGES SUPPLÉMENTAIRES
============================================================

─── /badges ────────────────────────────────────────────────

  Vitrine de tous les badges :
  - Grille de badges (débloqués en couleur, verrouillés en gris)
  - Cliquer sur un badge = voir sa condition
  - Barre de progression vers le prochain badge
  - Filtre par catégorie (progression, maîtrise, streak, fun)
  - Compteur : "12/47 badges débloqués"

─── /stats ─────────────────────────────────────────────────

  Page détaillée des statistiques :
  - Graphique d'activité (30 jours, style GitHub)
  - Graphique de progression par matière (courbe)
  - Temps passé par jour/semaine/mois
  - Meilleurs scores quiz
  - Taux de "Je sais" flashcards dans le temps
  - Historique des rangs atteints

─── /revision ──────────────────────────────────────────────

  Page "Révision du jour" dédiée :
  - File de flashcards à revoir (répétition espacée)
  - Quiz recommandés (chapitres faibles)
  - Exercices non terminés
  - Peut être lancée en un clic depuis l'accueil


============================================================
  RESPONSIVE / MOBILE
============================================================

Sur mobile, les sections s'empilent verticalement :

  1. Bandeau de bienvenue (compact)
  2. Révision du jour (priorité maximale)
  3. Objectifs de la semaine
  4. Reprendre où tu t'es arrêté
  5. Progression (accordéon par niveau)
  6. Accès rapide (grille 2×2)
  7. Activité + Stats (résumé)
  8. Badges (carrousel horizontal)

Le header mobile :
  - Menu burger pour la navigation
  - XP + streak toujours visibles (mini bandeau)
