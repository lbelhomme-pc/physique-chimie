## 🎯 Objectifs du cours

- Savoir différencier un **atome** d'une **molécule**.
- Savoir lire et écrire une **formule chimique** (ex: $H_2O$).
- Connaître les **modèles moléculaires** de base (couleurs des atomes).

---

## 1. Qu'est-ce qu'une molécule ?

<div class="definition-box">

### 📘 Définition

Une **molécule** est une entité chimique électriquement neutre constituée d'un **assemblage d'au moins deux atomes** liés entre eux.

</div>

Par exemple, le dioxygène ($O_2$) est formé de deux atomes d'oxygène.

---

## 2. La formule chimique

La formule contient :

- Les **symboles** des atomes présents (toujours une majuscule).
- Le **nombre** de chaque atome en **indice** (en bas à droite). Si un atome est seul, on n'écrit pas 1.

<div class="example-box">

### 📝 Analyse de $H_2O$

- **H** : 2 atomes d'Hydrogène
- **O** : 1 atome d'Oxygène
- **Total** : 3 atomes

</div>

<div class="box-regle-or">

### ⚠️ Attention

Ne pas confondre **$Co$** (un atome de Cobalt) et **$CO$** (une molécule : 1 Carbone + 1 Oxygène). **La majuscule fait toute la différence !**

</div>

---

## 3. Les modèles moléculaires

Code couleur international :

| Atome | Symbole | Couleur |
|---|---|---|
| Carbone | $C$ | **Noir** |
| Hydrogène | $H$ | **Blanc** |
| Oxygène | $O$ | **Rouge** |
| Azote | $N$ | **Bleu** |

---

## 📌 L'essentiel à retenir

- Une **molécule** = assemblage d'au moins deux atomes.
- La **formule chimique** donne les symboles et le nombre d'atomes.
- Le chiffre **1** ne s'écrit jamais.
- Attention aux **majuscules** : $Co \neq CO$.
- Code couleur : C = noir, H = blanc, O = rouge, N = bleu.
